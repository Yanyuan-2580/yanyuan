# 7 - 版本更新日志

> **项目**: 心理健康AI助手
> **当前版本**: v2.7.0
> **更新日期**: 2026-06-02

本文档面向用户，以简明易懂的方式记录每个版本的更新内容。

---

## v2.7.0 (2026-06-02) — 全局UI美化

- **关键词**: 视觉焕新 / 设计统一 / 治愈体验升级
- 全站 26 个页面统一采用"极简治愈轻奢风"视觉体系，雾凇青主色调搭配翡翠绿渐变，低饱和度色彩减轻视觉负担。
- 所有按钮统一为 calming → emerald 渐变风格，hover 时柔和上浮，交互反馈更加细腻。
- 导航介绍页（/nav）全新重写：Hero 大标题 + 呼吸光效背景 + 6 大功能卡片悬浮动画 + 滚动淡入动效，未登录用户体验大幅提升。
- 登录/注册页视觉升级：左侧品牌展示区 + 毛玻璃卡片 + 柔焦插画，对标专业心理健康平台品质感。
- AI 对话页聊天气泡圆角加大，消息渐入动画，输入框聚焦光效，对话体验更加温暖。
- 情绪日记、知识库、冥想放松、心理测评等所有页面全面优化圆角、间距、色彩和动效。
- 底部导航栏、页面标题栏、文章卡片、心情选择器等 8 个核心组件视觉升级，选中态增加光效反馈。
- 新增空状态友好提示（心理安抚语气），告别冰冷的"暂无数据"。

## v2.6.0 (2026-05) — API导出修复 + 视频页完善

- **关键词**: 视频通话 / 系统稳定性 / 登录体验优化
- 视频咨询页面功能完善：支持创建加入视频房间、文字聊天侧边栏、参与人列表，为后续视频通话功能打下基础。
- 忘记密码流程优化：密码重置成功后自动登录，无需重复输入账号信息。
- 验证码登录体验优化：输入框自动聚焦，发送按钮增加 60 秒倒计时，防止重复点击。
- 修复了部分 API 接口的导出问题，提升系统稳定性。
- TypeScript 类型定义补全，代码质量进一步提升。

## v2.5.1 / v2.5.0 (2026-05) — 导航介绍页

- **关键词**: 项目导航 / 未登录体验 / 功能一览
- 新增项目导航介绍页（/nav）：未登录用户进入应用后先看到完整的项目介绍，而非直接跳转登录页。
- 导航页展示 6 大核心功能：AI 对话陪伴、情绪日记、知识文库、冥想放松、心理测评、视频咨询。
- 展示平台特色亮点：专业 AI 模型驱动、隐私安全保障、风控防护体系。
- 底部引导注册登录，降低新用户的心理门槛。
- 修复了移动端导航页的响应式布局问题。

## v2.4.0 (2026-05) — 登录注册页大改造

- **关键词**: 登录页焕新 / 品牌形象 / 简单心理风格
- 登录页和注册页全面视觉升级，对标专业心理健康平台"简单心理"的设计品质。

## v2.3.0 (2026-05) — 忘记密码 + 验证码登录 + 用户设置

- **关键词**: 密码找回 / 验证码登录 / 账号安全
- 新增忘记密码功能：通过手机号或邮箱接收验证码，三步完成密码重置。
- 新增短信验证码登录：无需记忆密码，输入手机号和验证码即可快速登录。
- 用户设置页面完善：支持编辑头像、昵称、性别、生日、个性签名等个人资料。
- 新增帮助中心页面：常见问题 FAQ，解答使用中的疑惑。
- 新增关于页面：展示项目介绍、版本信息和团队信息。
- 验证码有效期 5 分钟，同手机号每分钟限发 1 次，保障账号安全。

## v2.2.0 (2026-05) — 风控系统 + 通知 + 评论

- **关键词**: 安全保障 / 消息通知 / 社区互动
- 上线智能风控系统：自动检测对话、日记、评论中的风险内容（覆盖 35+ 高危词和 15+ 中危词），及时预警干预，守护用户安全。
- 上线消息通知系统：支持系统通知、评论回复通知、风险提醒、测评结果通知等多种类型。
- 未读通知提醒：底部导航栏显示未读消息红点，每 30 秒自动刷新，不错过重要消息。
- 文章评论区功能上线：支持评论和回复，形成知识社区的互动讨论氛围。
- 评论内容同样经过风控检测，确保社区环境安全健康。

## v2.1.0 (2026-05) — 管理后台

- **关键词**: 后台管理 / 数据统计 / 内容运营
- 管理后台正式上线（管理员专属），为运营团队提供强大的管理工具。
- 仪表盘实时展示用户总数、今日活跃数、新增日记数、AI 对话次数等关键数据。
- 用户管理功能：查看所有用户列表、搜索筛选、启用/禁用账号，高效管理用户。
- 文章管理功能：发布、编辑、审核文章，管理知识分类，确保内容质量。
- 风控管理功能：查看风险记录、标记处理状态，及时跟进异常情况。
- 管理员操作日志自动记录，所有操作可追溯。

## v2.0.0 (2026-05) — 五大模块扩展

- **关键词**: 心情追踪 / 冥想放松 / 心理测评 / 视频咨询 / 功能大扩展
- 心情追踪模块上线：每日记录情绪状态，支持 10+ 种情绪类型和强度选择，查看情绪变化趋势图表，了解自己的心理状态变化。
- 冥想放松模块上线：提供呼吸练习、身体扫描、正念行走等多种冥想课程，内置冥想计时器和完成记录，帮助用户养成冥想习惯。
- 心理测评模块上线：提供多套专业心理测评问卷，覆盖焦虑、抑郁、压力等常见心理状态评估，完成测评后获得 AI 生成的个性化分析建议。
- 视频咨询模块上线：支持创建和加入视频咨询房间，为未来远程心理咨询服务打下基础。
- 个人中心功能完善：新增通知列表、提醒设置、帮助中心和关于页面。

## v1.3.0 (2026-04) — 知识库模块

- **关键词**: 心理知识 / 文章阅读 / 收藏点赞
- 知识库模块上线：提供丰富的心理健康科普文章，涵盖焦虑管理、情绪调节、人际关系、自我成长、睡眠改善、正念冥想、心理科普等 7 大分类。
- 文章搜索和分类筛选：快速找到想看的内容。
- 一键点赞和收藏：喜欢的文章随时收藏，方便反复阅读。
- 文章阅读计数：了解热门文章动态。

## v1.2.0 (2026-04) — 情绪日记模块

- **关键词**: 日记记录 / 情绪标签 / AI 分析
- 情绪日记功能上线：支持创建、编辑、查看和删除个人日记。
- 心情标签功能：记录每篇日记时选择当时的心情状态（开心、平静、焦虑、悲伤、愤怒、疲惫、期待、满足、迷茫、感恩）。
- AI 心情分析：AI 自动分析日记内容，识别情绪状态并给出暖心建议。
- 情绪趋势统计：通过折线图直观查看近期情绪变化趋势，更好地认识自己的心理状态。

---

## v1.1.0 (2026-04) — 流式对话 + 多会话

- **关键词**: 流畅对话 / 多会话管理 / 打字机效果
- AI 对话升级为流式响应：AI 的回复像真人打字一样逐字显示，对话体验更加自然流畅。
- 多会话管理：支持新建多个对话主题，自由切换不同的聊天会话，历史会话随时回顾。
- 会话重命名和删除：方便整理和管理对话记录。
- 网络异常自动重试：SSE 流式连接意外中断时自动恢复，不影响对话体验。

## v1.0.0 (2026-04) — 项目正式启动

- **关键词**: 项目诞生 / AI 对话 / 用户系统
- 心理健康AI助手项目正式启动，致力于为用户提供便捷、专业的在线心理健康服务。
- AI 对话陪伴功能上线：与 AI 助手进行心理倾诉和对话，获得情绪疏导和心理支持。
- 用户注册和登录系统：支持手机号注册，安全登录保护个人隐私。
- JWT 安全认证机制：采用行业标准的双 Token 认证方案，保障账号和数据安全。
- 简洁清新的用户界面：初见"极简治愈"的设计风格，营造舒适的使用氛围。
